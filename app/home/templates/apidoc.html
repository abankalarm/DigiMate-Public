Api Documentation
This section deals with several apis that can be integrated in other platforms to increase functionalities.


x-access-tokens - for all authenticated API endpoint, this token needs to be supplied
valid token is Base64 encoded version of USERNAME:PASSWORD
authenticated APIs

Get your own personal details

/api/profile/own - this api lets you retrieve all your profile data
Example:
curl --header "x-access-tokens: SFIyMDY5OkhSMjA2OQ==" http://127.0.0.1/api/profile/own


Tasks List

/api/task/list - this api lets you list all your active tasks.
Example:
curl --header "x-access-tokens: SFIyMDY5OkhSMjA2OQ==" http://127.0.0.1/api/task/list

Tasks add

/api/task/list - this api lets you add a task to your active tasks.
Example:
send post with todoitem param

Tasks List

/api/task/delete/{id} - this api lets you delete active tasks.
Example:
curl --header "x-access-tokens: SFIyMDY5OkhSMjA2OQ==" http://127.0.0.1/api/task/delete/{id}

Get courses list

/api/getcourse - this api lets you get course details from sites like udemy and course era for any specified query.

Example:
curl --header "x-access-tokens: SFIyMDY5OkhSMjA2OQ==" http://127.0.0.1/api/getcourse?query=data
